is an application that includes live train data projected on a map to track trains live. The application also offers data about the train itself like next stop and actual time of arrival. The app also offers an augmented reality version for the user to see the train and in 3D.

We are a collection of students from Plymouth University studying Internet Design. When we set out to do this project we thought of where this application could go, we came up with having this application in a big tablet in train stations to show travellers when their trains will arrive in a visual way instead of just times on a departure board.
